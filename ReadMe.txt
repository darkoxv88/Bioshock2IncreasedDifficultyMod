*How to install this mod*

Here is what you have to do to get this mod to work correctly.

1) Go to your Bioshock 2/SP/Content directory.

2) Make a copy of you System Folder, ConfigINI.ibf, and Localizedint.lbf, incase you want to remove this mod or it would cause problems running the game.

3) Rename ConfigINI.ibf to ConfigINI.ibf.bak and move Localizedint.lbf to the System Folder.

4) Drag the contents of the System Folder of this RAR file into your Bioshock 2/SP/Content directory, and if it asks to overwrite, click overwrite everything.

5) Now would you kindly start a new game since it is required to make this mod work correctly.
